Virtual Seating Allocation System
Overview

The Virtual Seating Allocation System is a web-based application that automates the process of assigning seats to students for examinations or classrooms. It helps reduce manual work, avoids seating conflicts, and efficiently utilizes available classrooms.

Features
Student Management
Classroom Management
Automatic Seat Allocation
Admin Login
View Seating Arrangement
Simple and User-Friendly Interface
Technologies Used
HTML
CSS
JavaScript
PHP
JSON
How to Run
Install XAMPP.
Copy the project folder to htdocs.
Start Apache.

Open:

http://localhost/Virtual-Seating-Allocation-System/
